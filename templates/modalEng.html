Welcome to the CardChecker!

This app allows you to check if your card is not fake and if it is one of the following:

  • American Express
  • MasterCard
  • Visa

The test is based on Luhn's algorithm and the whole operation is conducted in backend, in Python. The app has been built with a use of the following technologies:

  • Flask
  • Python
  • JavaScript
